# Timelapse Machine (ELEC 3300)

#robotics #embedded #stm32 

This robot is built for capturing time-lapses—long processes like plant growth condensed into short videos. Unlike a static time-lapse, it creates motion time-lapses, where the camera moves while recording. The robot takes tiny steps over long periods, producing smooth, dynamic shots. Its path can be set to follow a straight line, trace a circle, or track along a tape on the ground.

- Developed in C++ with STM32 platform
- Uses different components such as LCD, Steppers and infra sensors in GPIO and I²C
- Graded 34.6/40, with a mean of 21.5, SD 7.9
